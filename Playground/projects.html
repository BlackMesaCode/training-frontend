MyStack

MyStack is a prototype of a web based flash card service for students.

Features

  • Responsive design to support different devices
  • Different flash card query modes (e.g. random order, reverse)
  • Implemented leitner algorithm to make memorizing even more efficient
  • Statistics to monitor your learning process
  • Multi-lingual
  • Account management
  • Customizable import of csv flash cards from other flash card providers
  • XML import and export of flash cards to make backup and restore easy
  • Share cards with other students

Technologies used

  • C#
  • ASP.NET MVC
  • MSSQL and Entity Framework as ORM
  • HTML5/CSS3 frontend with css mainly relying on the back then brand new 'flex-box' model

Media Compass

Media Compass helps finding advertisers the right measures to support their campaign. With the help of an underlying data cube, for each measure a score is calculated based on the capaign criteria specified by the user. I continued the development of an already existing prototype when i startet my internship at Crossvertise GmbH in Munich.

Features

  • Covers wide range of measures from classic billboard advertisement all over to social media
  • Provide additional pro and contra arguments for each measure to make calculated score more transparent
  • Lots of tooltips that provide explanations to media related technical terms
  • Send results to mail

Technologies used

  • C#
  • ASP.NET MVC
  • HTML, CSS
  • AJAX
  • jQuery

BrainDump

BrainDump is a leightweight knowledge management tool, that tries to make structuring complex knowledge as easy as possible. It focuses on a fast and easy keyboard driven usage model, completely passing on distracting formatting features.

Features

  • Fast, lightweight and portable
  • Structure your knowledge in a hierarchical windows explorer-like way with the help of classic cut, copy and paste operations
  • Embedd web images with one hotkey
  • Full text search
  • XML import and export functionality to easily backup and restore data
  • Generate a text file from user data, which allows for easy printing
  • Included tutorial and usage tips

Technologies used

  • C# and WPF for GUI

Mubo

Mubo is a multi-instance botting system for World of Warcraft. It was used to automate various gameplay tasks, such as fighting or collecting ingame currency.

Technologies used

  • C# and WPF for GUI
  • Scripting language LUA (seperate ingame addon written in LUA to provide data to the external application)
  • Win32 API via PInvoke (mouse and keyboard-hook, scanning memory page tables, read&write memory addresses, manipulate window message loop, ..)

Features

  • Autonomous character navigation using waypoints and advanced pathfinding algorithms
  • Full combat automation
  • Define custom routines and assign them to hotkeys
  • GUI to define custom waypoint-sets
  • Capable of running multiple instances simultaneously
  • Intelligent cross-instance communication to emulate teamplay
  • Inspection and debugging of values read from memory
  • Advanced combat log
  • Alternatively: Assisted combat mode (enhance hotkeys actions with custom logic)
